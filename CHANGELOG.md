## [1.0.0] — 2025-07-31

### Added
- Initial public release of FileSort Pro
- Lexicographical (A-Z) ascending sort of directory contents
- Interactive CLI session with animated spinner and styled prompts
- Non-interactive CLI mode via command-line arguments
- Hidden file detection and optional display (`--hidden`)
- Export sorted results to plain-text file (`--export`)
- Grouped output by first letter with Unicode file-type icons
- Human-readable file size display (B / KB / MB / GB)
- Robust error handling for invalid paths and permission issues
- Persistent log file (`filesort_pro.log`) with timestamped entries
- Comprehensive unit test suite (`tests/test_filesort_pro.py`)
- Full ANSI colour palette for terminal output
- `setup.py` for pip-installable packaging
- MIT Licence
